プライバシーポリシー / Privacy Policy

最終更新日: 2026-04-30

「帳」(以下「本アプリ」) は、生活の端っこに置いておける小さな共有ボードです。 本アプリは「静かに残る場」であることを大事にしており、扱う情報も最小限にとどめています。

このポリシーは、本アプリが何を集め、何を集めないか、集めたものをどう扱うかを率直に説明することを目的としています。

1. 提供者

  • 提供者: pnly
  • お問い合わせフォーム

2. 大切にしていること

  • 手元のもの: 書いたものや描いたものは、まずあなたの端末に保存されます。
  • 共有は暗号化: 誰かと共有するときも、内容は端末で暗号化してからサーバーに送られます。サーバー側で内容を読むことはできません。
  • 匿名で使える: メールアドレスや電話番号、氏名は登録時に必要ありません。
  • 広告も追跡もありません: 行動解析や広告のためにデータを使うことはありません。
  • 通知もありません: 戻ってきてもらうためのプッシュ通知は送りません。

3. 取り扱う情報

3.1 端末に保存されるもの

書いたテキスト、写真、手描き、ボードの並びといった、本アプリで作るものはあなたの端末に保存されます。 共有していないものはサーバーには送られません。

アカウントの鍵 (本人確認や暗号化に使うもの) も、あなたの端末の安全な領域 (Keychain) にだけ保管されます。

3.2 共有時にサーバーに送られるもの

ボードを誰かと共有しているときは、共有相手と同期するために、必要最小限の情報がサーバーに送られます。

  • ランダムに生成されたあなたのアカウント ID
  • 公開鍵 (本人確認と、共有相手との暗号化に使うもの)
  • 暗号化された投稿・コメント・画像・手描きデータ
  • 共有グループの構成情報 (招待や参加日時など)

投稿の中身、コメントの中身、写真や手描きの中身は、サーバーから読み取ることはできません。 これらはあなたとグループのメンバーの端末でだけ復号できます。

3.3 サーバー運用上のログ

不正利用対策と障害調査のため、アクセス時刻や IP アドレスなどの一般的なアクセスログがサーバー上に一定期間保管されます。 これらをあなた個人の趣向や行動の分析に使うことはありません。

3.4 取得しないもの

本アプリは次の情報を取得しません。

  • メールアドレス、電話番号、氏名、住所
  • 位置情報、連絡先、写真ライブラリ全体へのアクセス
  • 広告 ID やトラッキング ID
  • 行動解析、滞在時間、閲覧履歴

4. 利用目的

取得した情報は次の目的にのみ使います。

  • アカウントの本人確認
  • グループメンバー間でのデータの中継・同期
  • サービスの安定運用と不正利用対策

広告配信や利用者プロファイリングのための利用は行いません。

5. 第三者への提供

本アプリは利用者の情報を第三者に販売・貸与しません。 ただし、サービスを動かすために以下の事業者を利用しています。

  • Cloudflare (サーバー、データベース、ファイル保管)
  • Apple (App Store 経由の配布、端末側のセキュリティ機能)

これらの事業者は、本アプリの提供に必要な範囲でのみ情報を扱います。

6. 通知・トラッキングについて

  • プッシュ通知は送りません。
  • リエンゲージメント目的の通知も行いません。
  • 端末をまたいだトラッキング (App Tracking Transparency の対象となるもの) は行いません。

7. データの保管と削除

  • あなたが投稿やコメントを削除すると、共有相手の端末からも順次消えます。
  • アカウントの削除をご希望の場合は、後述のお問い合わせフォームからアカウント ID を添えてご連絡ください。サーバー側のあなたの情報を削除します。
  • 端末をリセットしたりアプリをアンインストールしたりすると、端末上に保管されていた鍵やデータは失われます。共有データを引き継ぐためのリカバリ手順をアプリ内でご案内しています。

8. お子様の利用について

本アプリは 13 歳未満のお子様を対象に設計されていません。 登録時に個人情報を取得しないため、年齢確認も行っていません。

9. セキュリティ

  • 共有する内容は、端末上で暗号化したうえでサーバーに送られます。
  • 通信は HTTPS で行われます。
  • パスワードは存在せず、端末の鍵によって本人確認を行います。

10. 利用者の権利

利用者は提供者に対して、保有しているサーバー側のデータの開示・訂正・削除を求めることができます。 後述のお問い合わせフォームから、アカウント ID とあわせてご連絡ください。

11. 準拠法

本アプリの利用に関しては、日本法を準拠法とします。

12. 改訂

機能の追加や運用変更に伴い、本ポリシーを改訂することがあります。 重要な変更があるときは、本アプリ内またはこのページの更新で告知します。
